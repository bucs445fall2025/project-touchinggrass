# All pieces

## **User Story**
- As a tetris player, I want the game to have every tetris piece, so that I can have a complete experience playing.
---

## **Acceptance Criteria**
- Pieces: S, Z, square, long, L, reverse L, T
- The game must have sprites and collision for each of these pieces.
- For illustrations of what these pieces look like, image search "tetris pieces"
